Starlight Programming Language

A lightweight scripting language designed to explore how programming languages work internally

Overview

Starlight is a lightweight server-side scripting language with its own CLI runtime. It is designed to help developers understand how interpreters work — from source code to execution.

Key Features

FeatureDescription
Simple SyntaxEasy to read and write
Interpreter PipelineLexer → Parser → Evaluator
CLI RuntimeRun scripts directly from terminal
ExtensibleDesigned for experimentation

Installation


npm install -g starlight-cli

Usage


starlight file.sl

Example


# Hello world
sldeploy "Hello, Starlight!";

FileRole

Why Starlight

Modern languages hide internal mechanisms. Starlight exposes them. It is built for learning, experimentation, and understanding interpreter design.

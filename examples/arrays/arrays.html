Starlight Arrays Guide

Learn array operations, slicing, and transformations in Starlight

1. Creating Arrays


# Create an array
let numbers = [1, 2, 3, 4, 5];

# Print array
sldeploy "Array: " + numbers;

2. Accessing Elements


# Define array
let fruits = ["apple", "banana", "cherry"];

# Access elements by index
sldeploy "First: " + fruits[0];
sldeploy "Second: " + fruits[1];

3. Array Slicing


# Create array
let arr = [10, 20, 30, 40, 50];

# Slice from index 1 to 4
let part = arr[1:4];

sldeploy "Slice: " + part;

4. Slice with Step


# Create array
let arr = [1, 2, 3, 4, 5, 6];

# Slice with step (every 2 elements)
let result = arr[0:6:2];

sldeploy "Step slice: " + result;

5. Adding and Removing Elements


# Create array
let arr = [1, 2, 3];

# Add element
push(arr, 4);

# Remove last element
pop(arr);

sldeploy "Array: " + arr;

6. Map (Transform Array)


# Create array
let nums = [1, 2, 3];

# Multiply each element by 2
let doubled = map(nums, (x) => x * 2);

sldeploy "Doubled: " + doubled;

7. Filter (Select Values)


# Create array
let nums = [1, 2, 3, 4, 5];

# Keep even numbers
let evens = filter(nums, (x) => x % 2 == 0);

sldeploy "Evens: " + evens;

8. Reduce (Aggregate Values)


# Create array
let nums = [10, 20, 30];

# Sum all values
let total = reduce(nums, (acc, val) => acc + val, 0);

sldeploy "Total: " + total;

9. Loop Through Array


# Create array
let arr = [5, 10, 15];

# Iterate through array
for item in arr {

    # Print each value
    sldeploy "Value: " + item;
}

10. Nested Arrays


# Create nested array
let arr = [[1,2], [3,4]];

# Access nested values
sldeploy "Value: " + arr[0][1];
